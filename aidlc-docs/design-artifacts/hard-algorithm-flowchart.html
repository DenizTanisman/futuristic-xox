2 · terminal_score
Kazanç → WIN−depth (erken kazan) · kayıp → depth−WIN (geç kaybet) · beraberlik → 0

3 · heuristic(s)
Statik tahmin: 30·tehdit + 1·ekonomi + 5·merkez (ben − rakip)
4 · TT PROBE (Zobrist)
Bu pozisyon daha önce ≥ bu derinlikte arandıysa: Exact→değeri döndür · Lower→α yükselt · Upper→β düşür · α≥β → döndür. Kayıtlı en iyi hamle = sıralama ipucu.

6b · Sıra rakibe geçti
Negasyon VAR — rakibin en iyisi benim en kötüm.

7 · best / α güncelle
best = max(best, score) · α = max(α, best) · en iyi hamleyi sakla
8 · ALPHA-BETA KESME
if α ≥ β { break } — rakip bu dalı zaten elemeli; kalan kardeş hamleleri aramaya gerek yok. Asıl hız buradan gelir.
9 · TT STORE
Süre dolmadıysa pozisyonu kaydet: best ≤ α₀ → Upper, best ≥ β → Lower, aksi → Exact. (Saatle kesilen arama TT'yi kirletmez.)
return best
